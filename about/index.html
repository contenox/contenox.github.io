About

Contenox is a local AI workflow CLI built to get things done — not explain things.

It started as a Kubernetes AI governance runtime. After talking to engineers who were actually trying to automate their workflows, we pivoted hard. Engineers don't want managed LLM infrastructure. They want a sharp tool that works in their terminal, respects their machines, and doesn't report home.

So we built that. A single binary. Persistent AI plans backed by SQLite. Real shell execution. A full-screen TUI. JSON-based workflow chains. Remote hook support so the model can call any HTTP API as a tool. And a cookbook of verified recipes so you don't start from scratch.

The CLI is and always will be free and open source (Apache 2.0). We make money by hosting specific hook services — managed RAG, Playwright, secure code sandboxes — that teams don't want to run themselves. Those are coming.
